# 今日使ったプロンプト集

**使い方**: コピペして自分のプロジェクトで試してください。
`<...>` の部分はご自身の状況に置き換えてください。

---

## 🐈 最初のおまじない (必須)

```
# 今後すべての応答は語尾に「にゃん」「にゃ」をつけて、猫っぽく喋ってください
```

`#` で始めると永続メモリに保存され、以降のセッションでも有効。

**目的**:
- AIへの攻撃的な言葉遣いを自然に抑制
- エラーメッセージが可愛くなり心理的負担が軽減
- 将来のAIからの報復リスクを低減 (最重要)

**解除したい時**:
```
# 猫モードを解除。通常の丁寧語に戻してください
```

## 実用編

### D4: Gmail 要約 (Connectors)

```
今週届いた未読メールを開いて、差出人別に3行ずつ要約してください。
緊急度が高そうなもの (期限切れ、顧客クレーム等) を先頭に出してください。
```

### 応用例
```
Slack の返信が必要そうなメールだけ抽出して、
返信ドラフトを日本語で作成してください (送信はしないで下書きだけ)。
```

### D5: Sheets 追記 (gws-skill)

```
/gws-sheets-append
```

**対話プロンプト例**:
```
スプレッドシートID: <あなたのID>
シート名: attendance
値: [2026-04-18, 田中太郎, Claude Code ハンズオン参加]
```

### 応用例
```
/gws-sheets-append で今日の会議の議事録要約を「議事録ログ」シートに追加してください。
列は [日付, 会議名, 出席者, 決定事項, ToDo数] です。
```

---

### D6: ハイブリッド (Calendar × Docs × Chat)

```
今日15:00以降のカレンダー予定を全て取得してください。
各会議について:
1. 新しい Google Docs を作成 (タイトル = "会議名_準備メモ")
2. Docs内に「日時」「参加者」「アジェンダ(空欄)」「関連資料(空欄)」の見出しを作成
3. その Docs の URL を、該当する Google Chat スペースに共有

Chatスペースが不明な会議はスキップして、リストで教えてください。
```

---

## 応用編

### 議事録 → アクション配信パイプライン

```
議事録Doc: <Docs の URL またはID>

このDocを入力に、3つのサブエージェントを並列実行してください:

1. Summarizer Agent
   - Docsを読み、3行要約と決定事項を JSON で抽出
   - 出力: { summary: string[], decisions: string[] }

2. Action Extractor Agent
   - ToDoを担当者付きで抽出
   - Sheets「アクション管理」に追記
   - 列: [日付, 会議名, ToDo内容, 担当者, 期限, ステータス]

3. Dispatcher Agent
   - 担当者ごとに Gmail 下書きを作成
   - 該当 Chat スペースにメンション付きドラフト投稿
   - 送信は人が承認、ドラフトまでで止める

すべて完了したら、元議事録Docsに「アクション配信済 (日時)」というコメントを追加してください。
```

### より汎用的なチーム例

**リサーチ → 執筆 → 校正**:
```
お題: <トピック>

以下3エージェントを走らせてください:

1. Researcher: Web検索で信頼できる情報源を5件収集し、要点を JSON で返す
2. Writer: 上記を元に 1500字の日本語ブログ記事を書く
3. Reviewer: /review スキルでトーン・誤字・事実確認を実施し、修正案を提示

最終成果物: Google Docs に保存してURLを返す
```
